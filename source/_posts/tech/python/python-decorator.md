## Python Decorator

註解器目前已經

函數的方式，分為有參數與無參數註解：

#### 無參數註解器

```python
@no_param_decorator
def some_function():
    ...
```

#### 有參數註解器

```python
@param_decorator(param="something")
def some_function():
	...
```
